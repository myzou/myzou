关于使用IDEA连接mysql后不显示表的解决方案

发表于 2019-08-26

转载链接：https://blog.csdn.net/u012834773/article/details/89893678

总结

简单来说，就是Database>点击设置，然后设置》Schemas》选中显示的表保存即可

详细

在更换了最新版的2019.1版本的IDEA后，连接mysql，在配置页面也指定连接mysql的数据库了，但是，连接成功后，这是什么鬼？

我百度了一下并没有找到解决方案，也困扰了我一段时间。但是我无意间尝试成功，这个问题完美解决，方法分享给大家

点击Data Source的图标进去

点击Schemas选项，勾选你需要显示在idea的库，然后Apply

OK，解决。
